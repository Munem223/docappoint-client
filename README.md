# DocAppoint - Doctor Appointment Manager

**Live Website:**    docappointclient.vercel.app

A modern, full-stack Doctor Appointment Booking System built with **React + Vite + Tailwind CSS** (Frontend) and **Node.js + Express + MongoDB** (Backend).

## ✨ Features

- ✅ Secure user authentication with JWT (Login / Register)
- ✅ Beautiful Home page with Swiper slider and Top Rated Doctors
- ✅ All Doctors page with search functionality
- ✅ Doctor Details page with booking modal
- ✅ Private Dashboard with My Bookings (CRUD operations) and Profile management
- ✅ Fully responsive design for mobile, tablet, and desktop
- ✅ Toast notifications for all actions (no default alerts)
- ✅ Protected private routes that work even after page reload
- ✅ Clean, modern UI following assignment design guidelines

## Tech Stack (Client)

- React + Vite
- Tailwind CSS
- React Router DOM
- Axios
- React Hot Toast
- Swiper.js
- Lucide React Icons
